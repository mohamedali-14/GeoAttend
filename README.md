# 📍 GeoAttend

**Smart Attendance System with Location & Selfie Verification**

GeoAttend is a modern attendance management system that uses GPS location tracking and optional selfie verification to ensure accurate, cheat-proof attendance marking for educational institutions.

---

## 🚀 Features

### Core Features
- **📍 GPS Location Tracking** - Students must be within a specified radius to mark attendance
- **📸 Selfie Verification** - Optional photo verification per session (professor chooses)
- **🔐 Random Check Codes** - Additional security layer with time-sensitive verification codes
- **📊 Real-time Dashboard** - Live attendance tracking and updates
- **📑 Export Reports** - Generate Excel and PDF attendance reports

### Professor Features
- Create sessions with custom verification settings
- Choose per session: GPS only, Selfie required, or both
- Review and verify/ reject student selfies
- View real-time attendance counts
- Export attendance reports (Excel/PDF)

### Student Features
- Mark attendance within geofenced location
- Take selfie when required by professor
- Receive push notifications for session start
- View attendance history
