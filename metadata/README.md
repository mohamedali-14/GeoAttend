# GeoAttend – Firebase Backend & Emulator Setup

This repository contains the backend scripts and Firebase configuration for the GeoAttend project.  
Use it together with the frontend code (located in `/frontend` if present).

## Prerequisites

- **Node.js** (v20 or later) – [Download](https://nodejs.org)
- **Java** (v21 or later) – [Download OpenJDK](https://adoptium.net/)
- **Firebase CLI** – Install globally: `npm install -g firebase-tools`
- **Git** – to clone the repository
